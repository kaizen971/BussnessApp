# Tutoriel utilisateur – Démarrage EAS (Entreprendre Avec Succès)

Bienvenue dans l'application EAS ! Ce tutoriel va vous guider étape par étape pour configurer votre espace de gestion.

## Étape 1 – Créer votre business
Depuis l'écran d'accueil, cliquez sur « Créer un business ».
Renseignez les informations suivantes :
- Nom du commerce
- Devise (CFA, €, $, etc.)
- Secteur d’activité (ex. salon de coiffure, restaurant…)
- Logo (facultatif)

Cliquez sur « Enregistrer » pour passer à l'étape suivante.

## Étape 2 – Ajouter votre équipe
Accédez au menu « Équipe ».
Cliquez sur « Ajouter un membre ».
Indiquez :
- Le prénom et le rôle (manager, caissier, coiffeur…)
- Son numéro ou email pour l’inviter
- Choisissez le niveau d’accès : lecture seule, ventes, admin

Cliquez sur « Enregistrer ».

## Étape 3 – Créer les catégories de produits/services
Dans le menu « Produits », allez sur « Gérer les catégories » (admin uniquement).
Créez vos catégories : soins, boissons, accessoires…
Enregistrez.

## Étape 4 – Ajouter vos produits ou services
Menu « Produits » → « Ajouter un produit ».
Remplissez :
- Nom
- Prix unitaire
- Catégorie
- Description (facultatif)

Enregistrez – répétez pour chaque produit/service.

## Étape 5 – Configurer le stock initial
Menu « Stock ».
Pour chaque produit, entrez la quantité disponible.
Enregistrez aussi les futures entrées/sorties (achats, ventes…).

## Étape 6 – Paramètres de vente
Allez dans « Paramètres » > « Ventes ».
Activez ou non :
- Retour / annulation
- Affichage historique aux employés
- Export / partage (ex. ticket client par WhatsApp)

Enregistrez.

## Étape 7 – Fidélité (facultatif)
Menu « Paramètres » > « Fidélité ».
Activez le programme.
Exemple : 1 point / 1 000 F dépensés.
Définissez les récompenses ou réductions.

## Étape 8 – Lancer votre activité
Vous êtes prêt à utiliser EAS :
- Enregistrez vos ventes
- Suivez vos revenus, charges, stock
- Gérez vos équipes et vos clients
- Exportez vos données (Excel, PDF)
